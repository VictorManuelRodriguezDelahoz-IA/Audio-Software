# Contribución

## Estructura de Documentación

### Docs editables (en Git)
- `Recochapp/docs/` - Documentación técnica de Recochapp
- `ML Studios/docs/` - Documentación técnica de ML Studios
- `docs/oportunidades-negocio/` - Investigación de oportunidades

### Docs sincronizados (desde Notion)
- `docs/startup-hq/` - Sincronizado automáticamente, **NO editar**

## Convenciones

### Commits
- `docs:` - Cambios en documentación
- `feat:` - Nueva funcionalidad
- `fix:` - Corrección de bugs
- `refactor:` - Refactorización

### Archivos de documentación
- Numerados: `01-nombre.md`, `02-nombre.md`
- Minúsculas con guiones: `modelo-negocio.md`

## Sincronización con Notion

```
Notion "Startup" page
      ↓ (Lunes 9 AM UTC o manual)
docs/startup-hq/STARTUP-HQ.md
```

Para editar contenido sincronizado, hazlo en Notion.
